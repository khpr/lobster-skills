# SP500 語言策略最終定版（2026-07）

## 核心原則
語言不是篩選標準。翻譯是基礎設施。

使用者為中文母語，所有非中文來源均透過 AI 翻譯摘要後閱讀。
唯一的來源評選標準：**翻譯後，這個觀點是否獨特且值得佔晨報版面？**

## 架構
- sources.md 每條加 `lang:en / lang:zh-tw / lang:zh-cn / lang:ja / lang:ko / lang:fr` 等標記
- L2 收割時保留語言標記
- 晨報產出時：非中文文章自動接 Gemini 翻譯，使用者收到的永遠是中文摘要

## 保留概念
- ZH-CN 加一個額外評審維度：編輯獨立性（文化/科技/財經數據可信，政治時事跳過）
- 各語言的獨特視角是價值，不是障礙
